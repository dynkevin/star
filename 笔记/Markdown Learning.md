## 一.基础

**我爱star**

## 二.列表

### 1.无序(* or- )

* Task1
  * Task1_1
  * Task1_2
* Task2
* ...

### 2.有序

1. Task1
2. Task2
3. ...

### 3.勾选(* [ ])

* [ ] 1
* [ ] 2
* [x] 3

